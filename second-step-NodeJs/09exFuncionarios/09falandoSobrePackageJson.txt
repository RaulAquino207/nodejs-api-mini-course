Arquivo referente ao um exemplo/projeto feito em NODE
Package.json = Arquivo descritor de um projeto em node 
Dentro da pasta 09exFuncionarios daremos um NPM INIT pelo terminal, habilita para se definir
como será seu arquivo package.json.
Caso queira responder sim para tudo faça NPM INIT -Y

Package.json ele descreve seu projeto, ou seja, ele tem descrito em seu corpo, 
todas as dependencias usadas no seu projeto

Quando usasse npm i ..., ele cria a node_modules dentro da pasta onde ele está sendo executado.
E além de instalar a dependencia dentro da pasta onde se está, para salvar dentro do package.json
da sua pasta usasse o comando npm i --save

No caso dessa aula o professor usou npm i --save axios, logo será encontrado dentro do package.json
dessa pasta uma versão da minha dependecia instalada, no caso o axios.
